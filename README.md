TRethinkWire
============

Delphi RethinkDB driver

RethinkWire.pas
  TRethinkWire: manages a connection to a [rethinkdb][1]

ql2.pas is generated using [DelphiProtocolBuffer][2]'s "dpbp" on rethinkdb's ql2.proto
